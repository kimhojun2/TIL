2. 

- cd . : 현재 디렉토리
- cd .. : 상위 디렉토리
- touch : 파일생성
- mkdir : 새 디렉토리 생성
- ls : 현재 작업 중인 디렉토리 내부의 폴더/파일 목록을 출력
- cd : 디렉토리 변경(위치이동)
- rm : 파일 삭제


Working Directory : 실제 작업 중인 파일들이 위치하는 영역

Staging Area : 중간 준비 영역

Respository : 영구적으로 저장되는 영역

- git init : 로컬 저장소 설정
- git add : staging area에 추가
- git commit : 저장소에 기록
- git config --global user.email "메일주소"
- git config --global user.name "유저네임"
- git status
- git local
- git reset :  add한거 취소
- git commit -m " 이름 "
